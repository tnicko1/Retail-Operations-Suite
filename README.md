# 📦 Retail Operations Suite

**A professional desktop application designed to streamline retail operations, from dynamic price tag generation to inventory management.**

This software is a powerful tool for retail store owners and managers who need to create accurate, multi-language price tags quickly and manage the status of items on display across multiple branches.

### ✨ Key Features

* **🏷️ Dynamic Price Tag Generator:** Create beautiful, accurate price tags in multiple sizes and styles.
* **🌐 Multi-Language Support:** Generate tags in both English and Georgian simultaneously. The UI is also fully bilingual.
* **📄 Dynamic Layout Engine:** Long item names and specifications automatically wrap to multiple lines, ensuring all information is displayed correctly without messy overlaps.
* **🎨 Customizable Themes:** Switch between different visual themes for your price tags, including a festive "Winter" theme with unique visual assets.
* **🏪 Branch & Display Management:** Track which items are on display in each store branch. A dedicated Display Manager helps you find replacement items from the same category when a tag is returned.
* **☁️ Database Integration:** Powered by a secure Firebase backend to sync product information, stock levels, and display statuses across the company.
* **📊 Admin Dashboard:** A comprehensive dashboard for administrators to view branch statistics, track low-stock items, and manage user activity.
* **🔄 Built-in Updater:** The application automatically checks for new versions upon startup to ensure you always have the latest features and bug fixes.
