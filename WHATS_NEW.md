# What's New

## Version 3.1.1
- **Enhanced Brand Selection:** The brand dropdown in the main window is now searchable, making it easier to find your desired brand.
- **Print Queue Improvements:**
    - Added a "Global Brand Override" option to the Print Queue dialog, allowing you to set a brand for all items in the batch.
    - Integrated "Automatic" brand detection directly into the queue logic, removing the need for a separate "Modern Design" checkbox.
    - Brand selection in the Print Queue is now searchable.
- **PCShop.ge Integration:** Improved the product scraper for pcshop.ge with better SKU-based search and anti-blocking measures.
- **Brand Updates:**
    - Added support for the **UGREEN** brand.
    - Enabled specifications display for the **Epson** brand.
- **Bug Fixes:** Resolved a permission error related to the templates file location.

- Initial release of the redesigned Retail Operations Suite.
